# kolesnik.io

Personal consulting site for Mykhaylo Kolesnik. Built with Astro, Tailwind CSS, and Biome.

## Local development

```sh
pnpm install
pnpm dev      # http://localhost:4321
pnpm check    # astro check + biome check
pnpm format   # apply safe fixes
pnpm build    # static build to ./dist
```

Requires Node 22.12+ and pnpm. See `.nvmrc`.
